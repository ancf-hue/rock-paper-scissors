# 🎮 Камень, Ножницы, Бумага

Классическая игра **Камень-Ножницы-Бумага** в современном исполнении! 

## 📱 О проекте

Бесплатная офлайн-игра, которая работает прямо в браузере. Никакой установки, никаких регистраций — просто откройте и играйте!

### ✨ Особенности:
- 🌐 **Работает без интернета** — установите как PWA приложение
- 🎯 **Простой и понятный интерфейс**
- 📊 **Статистика игр** — отслеживайте свои победы
- 🎨 **Современный дизайн** с неоновыми эффектами
- 📱 **Адаптивная версия** — играйте на телефоне и компьютере
- 🔒 **Полная конфиденциальность** — никаких данных не собирается

##  Как играть

1. Нажмите кнопку **"НАЧАТЬ ИГРУ"**
2. Выберите один из трёх вариантов:
   - 🪨 **Камень**
   - ✂️ **Ножницы**  
   -  **Бумага**
3. Сравните свой выбор с компьютером
4. Побеждайте и набирайте очки!

### Правила:
- 🪨 Камень бьёт ✂️ Ножницы
- ️ Ножницы режут 📄 Бумагу
- 📄 Бумага накрывает 🪨 Камень

##  Установка

### Вариант 1: Открыть в браузере
Просто перейдите по ссылке: **[anti-ai-artificial-intelligen.pages.dev](https://anti-ai-artificial-intelligen.pages.dev)**

### Вариант 2: Установить как PWA
1. Откройте сайт в браузере Chrome/Edge
2. Нажмите на значок **"Установить приложение"** в адресной строке
3. Или в меню браузера выберите **"Установить"**
4. Игра появится на рабочем столе и будет работать офлайн!

### Вариант 3: Скачать и запустить локально
1. Скачайте архив с проектом
2. Распакуйте в любую папку
3. Откройте файл `index.html` в браузере
